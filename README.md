# Planos-Estrat-gicos---Gerencie-seus-planos-de-carreira
Aplicação web para criação, edição e gerenciamento de Planos Estratégicos de Carreira. Permite registrar metas, planos de ação, análise SWOT, execução e indicadores de desempenho, tudo em uma interface moderna desenvolvida com React, Vite e Tailwind.
